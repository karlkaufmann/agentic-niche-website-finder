# Agentic niche website finder

Jupyter notebooks pro experimenty s agentním a LLM vyhledáváním webů v nikových tématech.

Toto veřejné repo obsahuje pouze notebooky a dokumentaci bez datových souborů a bez uložených modelů. Lokální výsledky a binárky modelů patří do `.gitignore` a nepatří do git historie.

## Obsah

- `Agentic website finder.ipynb`, `LLM website finder.ipynb`, `Agent Metrics.ipynb` — pracovní notebooky a metriky.

## Lokální spuštění

Použijte vlastní virtuální prostředí a závislosti podle vašeho workflow; `venv/` a `.venv/` nejsou součástí repozitáře.
